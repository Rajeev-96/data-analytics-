 Project Overview:-
The Smart Home Automation System is an IoT-based project that allows users to remotely monitor and control home appliances such as lights, fans, and sensors using a web or mobile interface. The system improves comfort, energy efficiency, and security by enabling real-time automation and remote access.

This project demonstrates practical implementation of Internet of Things (IoT), JavaScript, Node.js, and embedded systems concepts.
